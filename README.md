# Sales and Profit Dashboard

## Project Overview

This project focuses on analyzing sales and profit performance using Microsoft Excel. An interactive dashboard was created to visualize business performance, sales trends, profit distribution, and customer insights.

The project demonstrates data preparation, Pivot Table analysis, dashboard creation, and data visualization skills using Excel.

---

## Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

---

## Dataset Information

The dataset contains sales transaction records including:

* Order Date
* Customer Name
* State
* Category
* Sub-Category
* Product Name
* Sales
* Quantity
* Profit
* Month
* Year

---

## Data Preparation Steps

* Organized the raw dataset into tabular format
* Verified column consistency
* Formatted sales and profit values
* Created Month and Year columns for analysis
* Prepared the dataset for Pivot Table reporting and dashboard visualization

---

## Dashboard Features

* Total Sales KPI
* Profit Analysis
* Monthly Sales Trend
* State-wise Sales Analysis
* Category & Sub-Category Performance
* Top Customer Analysis
* Interactive Filtering using Slicers

---

## Business Insights

* Identified high-performing product categories
* Analyzed profit trends across different years
* Compared sales performance across states
* Tracked monthly sales fluctuations
* Identified top customers contributing to profit
